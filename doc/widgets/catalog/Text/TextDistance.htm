Text distance

Compute distances between documents.

Channels

Inputs

Examples (ExampleTable)
Attribute-valued data set.

Outputs

Distance matrix (SymMatrix)
Matrix where rows and columns are documents and values are distances between them.

Description

This is a simple widget that takes an ExampleTable with text features as metaatributes and computes the distances between documents based on those features. The Matrix type box is used to specify the output matrix type. If Similarity matrix is chosen, a larger value in the matrix means that two documents are more similar, whereas choosing the Distance matrix option means that a larger value will indicate greater distance between two documents.
